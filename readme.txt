descrição curta na tela inicio do produto

== Descrição ==

O plugin mostra as descrições inseridas dos produtos na página inicial, sem precisar entrar dentro do mesmo, assim o usuário podendo ler, mais produtos de uma vez,
sem precisar entrar em um de cada vez.

= Compatibilidade =

Requer WooCommerce 3.0 ou posterior para funcionar.


== Instalação ==

1. Carregue os arquivos do plug-in no diretório `/ wp-content / plugins / plugin-name` ou instale o plug-in diretamente na tela de plug-ins do WordPress.
2. Ative o plugin através da tela 'Plugins' no WordPress


== Frequently Asked Questions ==

= O que eu preciso para utilizar este plugin? =

* WooCommerce 3.0 ou posterior.

= Basta ativar o plugin e a descrição dos produtos ja aperece na página inicial? =

Sim, automaticamente a descrição já aparece na página inicial, sem nenhum tipo de configuração. Só ressaltando que aparece escrito o que você intruduzir na opção "Breve descrição sobre o produto".
